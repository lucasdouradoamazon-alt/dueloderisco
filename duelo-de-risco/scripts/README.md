# Scripts Python - Duelo de Risco

## Ambiente Virtual Ativo! ✅

O ambiente virtual está em `/venv`.

### Comandos:

```bash
# Ativar (se necessário)
.\venv\Scripts\activate

# Instalar dependências
pip install -r requirements.txt

# Desativar
deactivate
```
